Védett Állatok - Web-programozás I. Beadandó

Ez a projekt a Web-programozás I. előadás keretében készült házi feladat. A webalkalmazás célja a védett állatok bemutatása és különböző webes technológiák (JavaScript CRUD, OOJS, Fetch API, React) gyakorlati alkalmazása.

Elérhetőség
A működő alkalmazás megtekinthető az alábbi linken: http://webfeladat.nhely.hu/index.html

Alkalmazott technológiák
Frontend: React 19, JavaScript (ES6+), Axios és Fetch API, Bootstrap 5, CSS3
Backend: PHP 8, MySQL (PDO).
Eszközök:  Vite (Build folyamat), Git/GitHub (Verziókövetés)

Projekt felépítése
index.html: Központi menürendszer.
javascript.html: Alapszintű CRUD műveletek (tömb alapú tárolás).
react.html: React alapú adatkezelés.
fetchapi.html: JS + Fetch API + PHP (Adatbázis alapú tárolás).
axios.html: React + Axios + PHP (Adatbázis alapú tárolás).
spa.html: Single Page Application két React-alapú alkalmazással.
oojs.html: Objektumorientált JavaScript grafikus modul.
